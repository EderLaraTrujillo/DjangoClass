Primera Vez:
1: Copiar direccion del repositorio: https://github.com/<usuario>/DjangoClass.git
2: Configurar credenciales git: git config --global user.name "Nombre Propio"
			   	git config --global user.email "correo@correo.com"
3: git add . (punto)
4: git commit -m "Subiendo Repositorio"
5: git remote add origin https://github.com/<usuario>/DjangoClass.git
6: git push -u origin master --force
